# Eiker Energi – Next.js landing pages

Statisk markedsføringsside bygget med Next.js 14, Tailwind CSS og TypeScript.

## Kom i gang

```bash
npm install
npm run dev
```

## Bygg og test

```bash
npm run lint
npm run typecheck
npm run build
npm run export
npm run e2e
```

Nettsiden eksporteres statisk til `out/` og kan publiseres på GitHub Pages.
